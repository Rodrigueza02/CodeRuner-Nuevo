🎮 Sistema de Notificaciones - CodeRunner

📝 Instrucciones para Maria (UI/UX)

Este es un ejemplo visual de cómo deberían verse las notificaciones de logs narrativos.

Características:

  • ✅ Notificaciones en la esquina superior derecha
  • ✅ Colores según el tipo (error, advertencia, éxito, info)
  • ✅ Animaciones suaves de entrada y salida
  • ✅ Barra de progreso que indica cuándo se cerrará
  • ✅ Botón para cerrar manualmente
  • ✅ Modal centrado para mensajes importantes

Cómo Integrar:

  1. Copiar los estilos CSS a tu archivo de estilos
  2. Copiar las funciones JavaScript a tu código
  3. Escuchar el evento ui:mostrarLog desde PlayCanvas
  4. Llamar a mostrarNotificacion() cuando llegue un evento
